Nota: Las Interfaces pueden tomar el valor de clases que hayan heredado de la misma interface, de este modo se puede hacer el intercabio entre las implementaciones


@Autowired: Inyecta el componente. Puede ser en atributo,contructor รณ setter

@Qualifier: Especifica el componente a utilizar (se usa desde la inyeccion)

@Primary: Deja por defecto el componente (se usa en el componente)


@PostConstruct: Accion ejecutada luego de construir los componentes inyectados

@PreDestroy: Accion ejecutada antes de eliminar los componentes inyectados


@Scope: Ambito de la aplicacion, spring por default trabaja con singleton


Los Scopes en spring son:


  • Singleton
  • Prototype
  • Request
  • Session
  • Global session
  • Application
  • Websocket
